Team Roles – Smart Memory Glasses

Hardware & Embedded Systems

Handles integration of sensors (camera, mic, GPS), microcontroller programming, and hardware setup of the smart glasses.

AI Developer (Face Recognition)

Trains and implements real-time facial recognition models to identify known individuals using onboard camera input.

Backend & Database Developer

Responsible for managing the server side of the project, design and maintain the database, build backend logic.

Web Developer (Monitoring)

Builds a responsive dashboard for monitoring location, recognition logs, and sending alerts to caregivers via web.

Mobile App Developer

Develops a cross-platform mobile app for caregivers to receive real-time updates, alerts, and communicate with the system.

System Integrator & Tester

Ensures all modules work together, tests real-time data syncing, handles deployment, debugging, and system reliability.
